# ERASbook Cadastro de usuario 
Este cadastro Serve para cada usuário se cadastrar na plataforma com alguns campos de informações obrigatório 

## Campos necessarios

* Nome do usuário 

* E-mail 

* DDD/Telefone 

* CPF 

* Estado 

* Cidade 

* Escolher Plano 

* Escolher opção de pagamento: 

* Boleto bancário 

* PIX 

* Cartão de crédito
