# ERASbook Cadastro de Autor

Serve para o próprio autor cadastrar suas obras na livraria com alguns campos necessários 

## Campos necessarios

* Nome do usuário 

* E-mail 

* DDD/Telefone 

* CPF 

* Estado 

* Cidade 

* Tema literário 

* Escolher plano 

* Escolher opção de pagamento: 

* Boleto bancário 

* PIX 

* Cartão de crédito
